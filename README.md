# 🐍 Particle Snake — 光で描かれるスネークゲーム

“光の粒子でできたスネーク” を指・マウスで操作するミニゲームです。  
スコア（餌を食べた数）が増えるほど **スネークが長く & 太く成長**します。  
ネオンの光が尾を引くビジュアルで、見ているだけでも気持ち良いインタラクティブ作品です。

---

## 🎮 特徴

- 指やマウスで **スムーズに追従する**スネークのヘッド  
- 走った跡が粒子で残る **ネオングローの尾エフェクト**  
- 餌（オレンジの点）を食べると  
  - **スコアが増える**  
  - **体が長くなる**  
  - **太さ（幅）も少し成長する**  
- 成長するにつれて操作感や見た目が変化し、達成感がある  
- ブラウザだけで遊べる（スマホ/PC 両対応）
